

## Portfólio — Desenvolvedor Web

Site single-page com tema escuro e moderno, visual tech/sofisticado.

### Seções

1. **Hero** — Nome, título (ex: "Desenvolvedor Web Fullstack"), breve tagline e botão CTA para contato. Fundo escuro com efeito sutil de gradiente.

2. **Sobre mim** — Foto placeholder + parágrafo de apresentação. Layout lado a lado (imagem + texto).

3. **Skills / Tecnologias** — Grid de ícones/badges com as tecnologias (React, TypeScript, Node.js, etc.) com visual de cards translúcidos (glassmorphism).

4. **Projetos** — Grid com 6 cards de projeto, cada um com: imagem placeholder, título, descrição curta, tags de tecnologias usadas e links (demo/código). Efeito hover elegante.

5. **Contato** — Formulário simples (nome, email, mensagem) + links para redes sociais (GitHub, LinkedIn). Validação client-side com feedback visual.

6. **Footer** — Copyright e links rápidos.

### Design
- Paleta: fundo escuro (#0a0a0f), acentos em azul/roxo neon, texto claro
- Tipografia moderna (Inter/system fonts)
- Animações sutis de scroll com transições suaves
- Navegação fixa no topo com scroll suave para cada seção
- Totalmente responsivo

### Dados
- Todos os projetos e informações como dados placeholder editáveis em arrays no código
- Sem backend necessário nesta fase
