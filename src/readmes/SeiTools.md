# 🧰 SeiTools - Gestão de Materiais e Ferramentaria

Plataforma empresarial para controle de inventário, empréstimos e auditoria operacional, com dashboard analítico e gestão por perfil de acesso.

## 📸 Preview

![SeiTools - Login](/readmes/SeiTools1.png)
![SeiTools - Dashboard](/readmes/SeiTools2.png)

## 🚀 Funcionalidades

- Dashboard analítico com KPIs e gráficos de movimentação.
- Gestão de inventário com cadastro e controle de estoque.
- Controle de empréstimos e devoluções.
- Log de auditoria das operações críticas.
- Controle de acesso por perfil (incluindo área administrativa).
- Exportação e impressão de relatórios.

## 🛠️ Tecnologias

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Supabase
- Radix UI / shadcn-ui
- Recharts
- React Hook Form + Zod

## 💻 Execução Local

```bash
git clone https://github.com/GuilhermeCalixto1/PedidosDeMateriais.git
cd PedidosDeMateriais
npm install
npm run dev
```
