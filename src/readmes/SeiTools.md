## 📌 Sobre

O **SeiTools** foi desenvolvido para centralizar o controle de materiais e ferramentaria em ambiente corporativo.

O sistema integra operações do dia a dia com visão analítica para gestão, permitindo acompanhar saídas, devoluções, estoque crítico e histórico de ações com rastreabilidade.

---

## 🚀 Tecnologias

- **React 18** + **TypeScript**
- **Vite**
- **Tailwind CSS**
- **Supabase** (autenticação e dados)
- **Radix UI / shadcn/ui**
- **Recharts** (gráficos e indicadores)
- **React Hook Form + Zod** (formulários e validação)
- **XLSX + File Saver** (exportação)

---

## ⚙️ Funcionalidades

- **Dashboard Analítico** com cartões KPI e gráficos de movimentação.
- **Gestão de Inventário** com cadastro, edição e monitoramento de nível de estoque.
- **Controle de Empréstimos e Devoluções** com histórico e recibos.
- **Auditoria de Ações** para rastrear operações críticas no sistema.
- **Controle de Acesso por Perfil** com recursos administrativos restritos.
- **Exportação e Impressão** de relatórios para apoio operacional.

---

## 💻 Execução Local

```bash
# 1) Clonar repositório
git clone https://github.com/GuilhermeCalixto1/PedidosDeMateriais.git

# 2) Entrar na pasta
cd PedidosDeMateriais

# 3) Instalar dependências
npm install

# 4) Rodar em desenvolvimento
npm run dev
```
