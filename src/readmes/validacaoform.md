# 📝 Validador de Formulário com Regex

Formulário de cadastro com validação em tempo real usando JavaScript e Expressões Regulares (Regex), com foco em UX e integridade dos dados.

## 📸 Preview

![Validação de Formulário](/readmes/validacao.png)

## 🚀 Funcionalidades

- Validação de Nome, Senha, CPF e Data com Regex.
- Bloqueio de envio quando há dados inválidos.
- Feedback visual de erro.
- Auto foco no campo com problema.
- Interface Dark Mode.

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript
- Regex
