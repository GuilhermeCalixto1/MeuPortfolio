# 🦟 Game Mata Mosquito

Jogo interativo em JavaScript puro no qual o objetivo é eliminar mosquitos dentro do tempo limite sem perder as 3 vidas.

## 📸 Preview

![Mata Mosquito - Tela Inicial](/readmes/matamosquito.webp)
![Mata Mosquito - Em Jogo](/readmes/matamosquito2.webp)

## 🚀 Funcionalidades

- Geração randômica de mosquitos na tela.
- Cronômetro de partida com condição de vitória.
- Sistema de vidas com game over.
- Níveis de dificuldade (fácil, normal, difícil).
- Responsividade com ajuste ao tamanho da janela.

## 🛠️ Tecnologias

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6+)
- SessionStorage
