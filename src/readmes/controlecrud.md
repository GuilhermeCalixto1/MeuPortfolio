# 📦 Controle de Fornecedores (CRUD)

Aplicação para gerenciamento de parceiros e fornecedores com CRUD completo e integração com API REST real.

## 📸 Preview

![Controle CRUD - Tela 1](/readmes/crud.webp)
![Controle CRUD - Tela 2](/readmes/crud2.webp)

## 🚀 Funcionalidades

- Cadastro de fornecedores (Create).
- Listagem e consultas por nome/código (Read).
- Edição de dados cadastrais (Update).
- Remoção segura de registros (Delete).
- Tratamento de erros de rede.

## 🛠️ Tecnologias

- JavaScript
- Vite
- Axios
- Node.js
- CSS3

## 💻 Execução Local

```bash
git clone https://github.com/GuilhermeCalixto1/Projeto-CRUD.git
cd Projeto-CRUD
npm install
npm run frontend
```
