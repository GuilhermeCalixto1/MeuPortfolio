## 🔖 Sobre

Este projeto é uma **Calculadora Web** desenvolvida para praticar lógica de programação e estilização avançada com CSS Grid. O objetivo foi criar uma interface agradável (tema Dark/Purple) que se adapta a diferentes tamanhos de tela, garantindo usabilidade tanto no desktop quanto no mobile.

A lógica lida com operações aritméticas básicas, tratamento de erros (como divisão por zero) e formatação de casas decimais.

## ⚙️ Funcionalidades e Recursos

- [x] **🔢 Operações Básicas:** Soma, Subtração, Multiplicação e Divisão.
- [x] **📱 Design Responsivo:** Layout fluido que se ajusta a celulares e tablets (`@media query`).
- [x] **🎨 UI Moderna:** Uso de gradientes, sombras (`box-shadow`) e efeitos de hover.
- [x] **🛡️ Tratamento de Erros:** Detecta operações inválidas e exibe mensagem de "Erro".
- [x] **🧹 Limpeza:** Botão "Limpar" para resetar o cálculo rapidamente.
- [x] **📐 CSS Grid:** Organização eficiente dos botões em layout de grade.
