# 🧮 Calculadora Web

Calculadora funcional e responsiva para prática de lógica de programação e estilização moderna com CSS Grid.

## 📸 Preview

![Calculadora Web](/readmes/calculadora.webp)

## 🚀 Funcionalidades

- Operações básicas: soma, subtração, multiplicação e divisão.
- Tratamento de erros (como divisão por zero).
- Botão de limpeza de cálculo.
- Interface responsiva para desktop e mobile.
- Layout organizado com CSS Grid.

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript

## 💻 Execução Local

```bash
git clone https://github.com/GuilhermeCalixto1/calculadora-js.git
```

Abra o arquivo principal da calculadora no navegador.
